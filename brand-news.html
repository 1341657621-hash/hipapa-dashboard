防晒赛道：Q1销售额28.47亿元（-9.5%），喷雾逆势+40.6%成第二曲线

2026年Q1主流电商防晒销售额28.47亿元（同比-9.5%），但结构性分化显著：防晒喷雾逆势增长40.6%（销售额1.05亿元），成为第二增长曲线。婴童防晒+40.2%领跑，运动服+74.1%、女装+64%成为核心增长极。4月防晒电商总盘同比增长19%突破40亿元。100-200元价格带占销售额66%是利润核心。竞品动作：安热沙（资生堂）3月推出“蓝宝瓶”儿童防晒，主打“非纳米技术”+“小金盾”标志，线上教育+线下体验双向赋能。MISTINE第七代小黄帽搭载AI线粒体级抗光损科技。海龟爸爸2025年GMV 8.2亿，欧睿认证“亚洲第一婴童防晒品牌”。（防晒棒/水/粉饼/气垫）+ AI赋能（MISTINE线粒体抗光损、资生堂AI香氛喷雾）成破局关键。婴童防晒"小金盾"标识成焦点，监管趋严利好合规品牌。青眼情报6月29日发布《2026防晒趋势白皮书》：全球防晒化妆品市场2024年153.65亿美元→2030年212.28亿美元，中国防晒服市场2026年预计554亿元。婴童防晒CAGR约18%，养肤型防晒CAGR约25%。防晒棒在韩国市场份额突破6%，成为“便携补涂”代表剂型。质润推出防晒棒三款（小光盾/小紫盾/小奶盾），分别针对户外、敏感肌、儿童。海龟爸爸2025年GMV 8.2亿，欧睿认证亚洲第一婴童防晒品牌。

来源：魔镜洞察 / 任拓2026防晒全场景消费大数据 / 2026.04-06

来源：青眼情报 / 海龟爸爸SGS检测报告

防晒赛道竞品：MISTINE、贝德美、润本、袋鼠妈妈、一页、安热沙6大品牌快速起量

安热沙（资生堂）—“国际大牌+线下渠道”：3月推出「蓝宝瓶」儿童防晒，主打“非纳米技术”+“小金盾”标志。线上教育+线下体验双向赋能，国际大牌入局儿童防晒对海龟爸爸形成直接竞争压力。

MISTINE—“AI抗光损+王楚钦代言”：第七代小黄帽搭载AI线粒体级抗光损科技，续约王楚钦升级“全球首席防晒代言人”。AI从营销工具升级为产品研发核心驱动力。

贝德美—“高浓度蓝甘菊+德国经典舒缓专利”：占据儿童防晒一定份额，专注“96%高天然成分比”“快速切断红敏源头”。与海龟爸爸同价格带（100-200元）直接竞争。

润本—“Q1业绩亮眼+防晒气垫新品”：2026年春夏季推出儿童防晒保湿日霜、防晒气垫等新品，精准卡位夏日儿童防晒需求。成分党路线，与海龟爸爸同价格带直接竞争加剧。

袋鼠妈妈—“宝宝姑姑对抗紫外线”：天猫婴童防晒多大品牌，专注“无添加”“柔和不刺激”，在天猫加强实力广告投放。

一页（newpage）—“分龄护理+上线7个月销售额超千万”：上线仅七个月销售额超千万，2025年业绩超10亿，组却快速成长。专注“宝宝的第一瓶面霜”。

来源：各品牌官方 / 魔镜洞察 / 任拓 / 淘宝大学 / 2026.04-07

青少年护肤竞品：袋鼠妈妈、青青有口、青梓婴童、青萌护肤、海龟爸爸等多品牌纷纷入局

海龟爸爸—“小蓝瓶SGS检测领跑+全国首个青少年祛痘团标”：28天祛痘减少54.83%、闭口减少67.96%。已抢占“标准制定者”身份，具备定义品类话语权的战略优势。

袋鼠妈妈—“全新青少年护理系列”：挪动物园不只是婴童护肤，青少年护理是其传统优势领域。05后消费心智增长时被动惠及。

青青有口、青梓婴童、青萌护肤等新锐品牌：纷纷进入青少年护肤赛道，“以酶抗痘”比刷酸更温和成技术趋势，窗口期正在缩小。海龟爸爸已凭占“先发者”优势，但需拓展产品线和消费者覆盖。

青蛇王子—“国货老牌回归”：中国儿童护肤品牌老字号，新品线强化“青少年专用”概念，带着经典品牌的信任优势。

来源：青眼情报 / 海龟爸爸SGS检测报告 / 各品牌官方 / 2026.04-07

婴童护肤竞品：一页、戴可思、丝塔芬、顺峰妈妈、安唯伊、蔷诺娜宝贝等多品牌占据市场

戴可思—“年度销售额3.74亿领跑天猫婴童护肤，占比7.8%”：强化“成分党”心智，联合第三方实验室做产品测评。618策略聚焦爆品+潜力款组合，搜索卡位“儿童面霜”“婴儿洗护”等核心品类词。

一页（newpage）—“上线7个月销售额超千万，2025年超10亿”：云中生怡在婴童护肤领域的探索，“宝宝的第一瓶面霜”定位专注新生儿市场。

丝塔芬（Cetaphil）—“百年护肤品牌下沉婴童线”：贝德玛旗下，在儿童抗敏感、温和洗护领域强势。国际大牌品质背书、医学级认可，国货品牌的主要竞争对手。

顺峰婴童—“医学背景+专业降遭团队”：华润三九原生护肤品牌，医药企业背景给予消费者强信任。产品强调“安全性”和“医学级检测”，与海龟爸爸的“标准制定者”路线形成差异化。

安唯伊（Evereden）—“婴童胎脂面霜全网销售额第一”：获CIC双认证，以胎脂科技占据高端婴童抗敏感市场。线上教育+线下体验强化“高端进口”心智。

蔷诺娜宝贝—“抗敏感护肤+婴童专用”：贝泰妮旗下婴童线品牌，延续成人线“抗敏感”专业形象，强化“儿童皮肤屏障修护”。和海龟爸爸在“抗敏感”领域直接竞争。

来源：魔镜洞察 / 天猫榜单 / CIC灼识 / 各品牌官方 / 2026.04-07

跨界品牌与新兴威脾：Babycare、幼岚、moodytiger、薄荷健康等跨界入局

moodytiger—“儿童防晒服订单增长142%”：将防晒场景与亲子运动深度绑定，自研Breath超细纤维科技面料，凉感织进纱线。户外服饰品牌跨界入局儿童防晒，把防晒从“护肤品”扩大到“服饰品”。

Babycare—“婴童用品头部扩展护肤线”：从婴童用品跨界到婴童护肤，商品类扩展至护肤、防晒、面霜等，借助产品品类和渠道覆盖的优势，快速抢占市场。

幼岚—“国民国风婴童服饰+护肤”：中国风婴童服饰头部品牌，近年来开始探索婴童护肤和防晒领域，原有服饰渠道和粉丝基础提供冲浪支撑。

薄荷健康—“婴童婴幼天然护肤”：来自澳大利亚，婴婴级天然护肤品牌，强调“吃得进的护肤品”，在高端婴童护肤领域一加竞争对手。
